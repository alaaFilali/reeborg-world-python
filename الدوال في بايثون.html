الدوال في بايثون

لقد استعملنا الدوال في الدروس السابقة كثيرا وهي عبارة عن دوال مبنية في python اي كل ماتحتاجه لاستعمالها هو مناداتها باسمها كالدالة len و max و print ...الخ

واحدة من اكثر الدوال التي استعملناها هي دالة print ونعرف انها دالة بأن لديها اسم ثم زوج من الاقواس واحيانا نضع شيء داخل الاقواس او شيئين وفي دالة print نضع ما نريد طباعته اي string 


print("Hello")

وكل الدوال الاخرى تعمل بنفس الطريقة مثلا مع دالة len نعطيها string مثلا وترجع لنا طوله ويمكننا تخزينه في متغير 


length = len("Hello")

الان ماذا لو اردنا ان نصنع دوال خاصة بنا؟ ولفعل ذلك نستعمل الكلمة def اي اننا نعرف دالة او بالانجليزية defining ثم نعطيها اسم ثم نضع الاقواس واخر شيء هي نقطتين فوق بعض والتي تعني ان كل شيء بعدها وامامه فراغ ينتمي الى الدالة 


def my_function():
    print("Hello")
    print("Bye")

ولكن لو اشغل البرنامج الان لن يحدث شيء والسبب هو اننا يجب ان ننادي الدالة باستخدام اسمها والاقواس ولا نضع اي شيء داخلها لان الدالة الخاصة بنا لا تحتاج مدخلات 


def my_function():
    print("Hello")
    print("Bye")
